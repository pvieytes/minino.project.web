Getting started

Screencast

Minino - an Erlang web miniframework - Screencast.

Create the application dir

$ mkdir kitty
$ cd kitty

Get Minino

Download

Get the binary.

$ wget http://minino.openshine.com/m/bin/minino
$ chmod 744 minino

Build

Get the source and build it. 
$ git clone git@github.com:openshine/minino.git
$ cd minino
$ make

Create a Minino Application

$ ./minino create-app id=kitty

Run Minino Server

$ ./minino runserver [Port]

Port is an optional argument, it is 8000 by default.

Check minino running at http://localhost:8000
