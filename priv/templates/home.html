Minino

An Erlang web miniframework.

As simple as possible

Install erlang, download a binary (escript), type two commands and you have minino running!!

$ ./minino create-app id=kitty
$ ./minino runserver

Web miniframework

What is included?

  • dispatch rules
  • templates system
  • sessions

What is not included?

  • data model

Wirtten in Erlang

Erlang is a general-purpose concurrent, garbage-collected programming language and runtime system. The sequential subset of Erlang is a functional language, with strict evaluation, single assignment, and dynamic typing.
